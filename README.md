## 开发

克隆和安装：

```bash
git clone git@github.com:savoygu/nuxt-cnode
cd nuxt-cnode
npm i
```

运行开发服务器：

```bash
npm run dev
```

构建用于生产的应用程序:

```bash
npm run build
```

本地预览生产版本:

```bash
npm run preview
```
